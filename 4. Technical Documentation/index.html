Setting Up Database Credentials

When you first open the JAR File the following screen is shown:

Enter the username and password you use to access your mysql database. This will create a config file in the following location:

C:/Logistics_Manager/logistics_manager.config

There you can change the username and password in future.

Setting Up Database Tables

The application needs a database name "logistics_manager" to function. In addition to the database, the application needs the following tables:

  1. branches
  2. companies
  3. shippings
  4. users
  5. vehicles

You can easily create these tables with the help of the included SQL script. Just open your MySQL command line client and run the following command:

sql> source <PATH_TO_SQL_SCRIPT>

Registering New Company

First You need to register a new company. Open the JAR File and click on register. This will open the following window:

The Registration Screen has two tabs: One for registering a new company and other for registering a new branch. On the first tab enter the details of the new company to be created. This is the code that registers the new company.

String SQL = "INSERT INTO `logistics_manager`.`companies`(`Name`,`Location`) VALUES (?,?)";
Registering New Branch

On the previous screen you registered a new company. On that screen click on the second tab. This will open the following tab:

Enter the details of the new branch to be registered. This is the sql query that creates a new branch:

String SQL = "INSERT INTO `logistics_manager`.`branches`(`Name`,`Location`, `CMPID`) VALUES (?,?,?)";
