    Features

    The Logistics Manager software provides features which are essential for a Logistics Company. These include:

    • Creating and Modifying Orders and their status
    • Managing a record of all the vehicles owned by the company
    • Managing the movement of shipments from one branch to another

    Installation

    The installation process requires you to follow two steps:

    Setup Database Credential
    For this extract the downloaded file and run the JAR file. When you open the JAR File for the first time, a window will popup and will ask you for your credentials
    Setup Database and Tables
    The Software needs a database and a few tables in it to function. To create the database and tables simply run the included SQL Script.
